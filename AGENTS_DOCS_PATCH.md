# Parche recomendado para docs/agents/*.md

Aplicar manualmente en cada archivo dentro de `docs/agents/`.

Buscar esta sección:

```md
## Documentos que debe leer antes de trabajar
```

Reemplazar por:

```md
## Lectura mínima

Leer siempre:

- `AGENTS.md`
- `docs/AI_ROUTER.md`
- archivos directamente afectados por la tarea

Leer documentación adicional solo según indique `docs/AI_ROUTER.md` para el tipo de cambio.
```

Si querés conservar la lista vieja de documentos, moverla debajo de:

```md
## Documentos relacionados

```
